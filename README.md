# Machine Failure Prediction (Predictive Maintenance)

ML model that predicts industrial machine failure using sensor data 
(temperature, torque, tool wear) from the AI4I 2020 Predictive 
Maintenance Dataset (UCI Machine Learning Repository).

**Results:** 98.15% Accuracy | 89.74% Precision (Random Forest Classifier)

**Tech Stack:** Python, Pandas, NumPy, Scikit-learn
